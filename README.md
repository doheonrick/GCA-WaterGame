# Refill Rush | charity: water Game Prototype

Refill Rush is a simple interactive browser game inspired by charity: water’s mission to bring clean water to people around the world.

In this prototype, players collect yellow water cans to earn points while avoiding polluted drops that reduce lives. The game is designed as a fast-paced, beginner-friendly HTML, CSS, and JavaScript project focused on core gameplay mechanics.

## Project Goal

The goal of this project is to create a working interactive game prototype that includes:

- Clickable game elements
- Score tracking
- A countdown timer
- Player lives
- Basic feedback messages
- A results screen
- A reset / replay option

This version focuses on functionality first, with visuals and polish to be improved in later iterations.

## Game Concept

The player has 60 seconds to collect as many clean water cans as possible.

### Rules
- Click the **yellow water cans** to earn points.
- Avoid the **polluted drops** because they reduce your lives.
- The round ends when:
  - the timer reaches 0, or
  - the player loses all lives.

## Features

- Start screen with instructions
- Live score display
- 60-second countdown timer
- 3-life system
- Combo tracking
- Falling clickable items
- Penalty obstacle
- End-of-round results screen
- Play again button
- charity: water inspired branding and colors
- Responsive layout for different screen sizes

## Built With
